# 📸 Instalike - Front-end

Este é o repositório do front-end do **Instalike**, uma rede social desenvolvida durante a Imersão Alura. O projeto permite o upload de imagens e gera descrições automáticas utilizando Inteligência Artificial.

## 🚀 Link do Projeto
Você pode acessar o site ao vivo aqui: https://instalike-frontend.vercel.app

## 🛠️ Tecnologias Utilizadas
- **HTML/CSS/JavaScript** (ou React/Vite, se for o caso)
- **Vercel** para o Deploy do Front-end

## ⚙️ Backend e API
Este front-end consome uma API personalizada que lida com o processamento de imagens e integração com o Google Gemini.

🔗 **Confira o repositório do Backend aqui:** https://github.com/kaykeoldenbrock/instalike-back

## 🌟 Funcionalidades
- Galeria de posts persistente no MongoDB.
- Upload de imagens reais.
- **Integração com IA:** Descrições geradas automaticamente pelo Google Gemini.
